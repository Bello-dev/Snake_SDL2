## üöÄ **INSTALLATION & SETUP**

### **Prerequisites**
```bash
# Ubuntu/Debian
sudo apt update
sudo apt install libsdl2-dev libsdl2-image-dev libsdl2-mixer-dev libsdl2-ttf-dev libsdl2-gfx-dev

# Arch Linux  
sudo pacman -S sdl2 sdl2_image sdl2_mixer sdl2_ttf sdl2_gfx

# macOS (Homebrew)
brew install sdl2 sdl2_image sdl2_mixer sdl2_ttf sdl2_gfx
```

### **Quick Start**
```bash
git clone https://github.com/Bello-dev/Snake_SDL2.git
cd Snake_SDL2
make clean && make
make run
```

### **Build Options**
```bash
make              # Standard build
make debug        # Debug build with symbols
make clean        # Clean build files
make help         # Show all options
```
